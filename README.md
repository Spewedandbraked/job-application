Перед установкой:
- Убедитесь, что порты 80 и 5432 свободны
- Создайте .env файл (скопируйте .env.example в .env)

Локальная установка:
1. docker compose -f compose.dev.yaml up --build -d

После установки приложение доступно по http://localhost/
